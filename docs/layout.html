Layout

To simplify the process of developing Web UI, Bootwrap uses the predefined layout which consists of two elements Page and Menu .

Note: if your application does not require the top-level menu it can be discarded.

Page

Constructor: Page(favicon=None, title=None, resources=None, menu=None, content=None)

Name Type Description
favicon str the file name for the favorite icon displayed in a browser tab;
title str the page title, displayed in a browser tab;
resources list the The list of Link s and Javascript s to be loaded with the page;
menu Menu the page top-level menu;
container WebComponent the WebComponent for hosting page elements;

The container could be any element inheriting WebComponent class. Typically for hosting other custom interface elements is used Panel . 

from bootwrap import Page

page = Page(
  favicon=..., title=..., resources=..., menu=...,
  container=...
)

Menu

Constructor: Menu(logo=None, brand=None, anchors=None, actions=None)

Name Type Description
logo Image the Image representing a company or organization logo;
brand Text the Text representing a company, organization, or project name;
anchors list the list of Anchor for navigation to the different pages from the top-level menu;
actions list the list of Button for performing top-level actions such as login, logout, etc;

The menu represents the top-level navigation bar containing anchors and actions allowing to switch between different application Page s. 

from bootwrap import Menu

menu = Menu(
  logo=..., brand=..., anchors=...,  actions=...
)
