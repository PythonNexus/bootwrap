Anchor

Constructor: Anchor(name)

Name Type Description
name str the anchor name

An Anchor can be used for creating a hyperlink to pages, files, email addresses, locations on the same page, or other web-resources defined by a URL address. In addition, it is also used in conjunction with other web-components, for example, in creating a navigation menu.

Use the link(href) function to connect an Anchor to another web-resource defined by the href parameter. 

Anchor('Search').link('https://www.google.com/')
Search

Anchor Styles

Use the following functions as_primary() , as_secondary() , as_success() , as_warning() , as_danger() , as_info() , as_light() , and as_dark() for changing an Anchor appearance.

Follow the Bootstrap guideline to applying these styles. 

Anchor('Primary').as_primary()
Anchor('Secondary').as_secondary()
Anchor('Success').as_success()
Anchor('Warning').as_warning()
Anchor('Danger').as_danger()
Anchor('Info').as_info()
Anchor('Light').as_light()
Anchor('Dark').as_dark()
Primary Secondary Success Warning Danger Info Light Dark

Button

Constructor: Button(name)

Name Type Description
name str the button name

A Button is a simple component for performing interactions on a website. Use the link(href) function to connect a Button to another web-resource defined by the href parameter. In this scenario, a Button behaves the same as an Anchor . 

# the button without style
Button('Search').link('https://www.google.com/')
Search

Button Styles

Use the following functions as_primary() , as_secondary() , as_success() , as_warning() , as_danger() , as_info() , as_light() , and as_dark() for changing a Button appearance.

To make a Button displayed without background call function as_outline() . Note, a Button will have an outline look only if you apply a specific style. For a Button without style, this function will have no effect.

Follow the Bootstrap guideline to applying these styles. 

# to make a different appearance
Button('Primary').as_primary()
Button('Secondary').as_secondary()
Button('Success').as_success()
Button('Warning').as_warning()
Button('Danger').as_danger()
Button('Info').as_info()
Button('Light').as_light()
Button('Dark').as_dark()

# to make the outline look
Button('Primary').as_primary().as_outline()
Button('Secondary').as_secondary().as_outline()
Button('Success').as_success().as_outline()
Button('Warning').as_warning().as_outline()
Button('Danger').as_danger().as_outline()
Button('Info').as_info().as_outline()
Button('Light').as_light().as_outline()
Button('Dark').as_dark().as_outline()

Form

Constructor: Form(*components)

Name Type Description
components list the list of input components and actions.

A Form is a web-component containing interactive controls for submitting information. 

Form(
  # here should be a list of input components and actions 
  ...
).on_submit('go/to/this/url')

Checkbox Input

Constructor: Checkbox(name, label)

Name Type Description
label str the input label showing for user
name str the input name, used for getting an input value from the form, during handling a submit action

A CheckboxInput is a component which can be in two states checked or unchecked.

The function check(status) changes the component state. The status parameter equals to True sets this component as checked and False as unchecked accordingly. A default value the status parameter is True .

Use the as_disabled() function to prevent user to modify a status of this component. 

CheckboxInput('One', 'opt1')
CheckboxInput('Two', 'opt2').as_checked()
CheckboxInput('Three', 'opt3').as_disabled()

Redio Input

Constructor: RedioInput(name, label)

Name Type Description
label str the input label showing for user
name str the input name, used for getting an input value from the form, during handling a submit action (this value should be the same for every component belonged to the same group)

A RadioInput is a component for creating a group of options. An individual RadioInput can be in two states, selected or not selected. Note, If one option is selected, all others automatically deselected.

The function with_value(value) assigns a unique option value , that set to the group when user selects a RadioInput component.

The function as_checked() changes the component state from deselected to selected .

Use the as_disabled() function to prevent user to modify a status of this component. 

RadioInput('One', 'choice').with_value('1')
RadioInput('Two', 'choice').with_value('2').as_checked()
RadioInput('Three', 'choice').with_value('3').as_disabled()
