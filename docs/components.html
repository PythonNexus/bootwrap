Class Anchor

A web-component for an anchor. 

Anchor(inner=None, role=None)
Arguments
Name Type Description
inner obj The str or WebComponent object wrapped by the anchor (default=None).
role str The anchor role. This parameter is not used in a general scenario. It usually set by other WebComponent , which is using the anchor to introduce a specific action (default=None).

An Anchor is used for creating a hyperlink to pages, files, email addresses, locations on the same page, or other web-resources defined by a URL address. The Bootwrap also uses the Anchor in conjunction with other components, for example, in creating a navigation menu.

Class Button

A web-component for a button. 

Button(name)
Arguments
Name Type Description
name str The button name.

Button Styles

Use the following functions as_primary() , as_secondary() , as_success() , as_warning() , as_danger() , as_info() , as_light() , and as_dark() for changing the Button appearance.

Use the as_outline() function to make the Button without background. Note, the Button will have an outline look only if you apply one of the predefined styles. For the Button without style, this function will have no effect.

Follow the Bootstrap guideline to applying these styles. 

# to make a different appearance
Button('Primary').as_primary()
Button('Secondary').as_secondary()
Button('Success').as_success()
Button('Warning').as_warning()
Button('Danger').as_danger()
Button('Info').as_info()
Button('Light').as_light()
Button('Dark').as_dark()

# to make the outline look
Button('Primary').as_primary().as_outline()
Button('Secondary').as_secondary().as_outline()
Button('Success').as_success().as_outline()
Button('Warning').as_warning().as_outline()
Button('Danger').as_danger().as_outline()
Button('Info').as_info().as_outline()
Button('Light').as_light().as_outline()
Button('Dark').as_dark().as_outline()

Class Link

A web-component for an external resource link. 

Link(href, rel='stylesheet', ctype='text/css')
Arguments
Name Type Description
href str This attribute specifies the URL of the linked resource. A URL can be absolute or relative (from MDN WebDoc).
rel str This attribute names a relationship of the linked document to the current document (from MDN WebDoc).
ctype str This attribute is used to define the type of the content linked to (from MDN WebDoc).

See https://developer.mozilla.org/en-US/docs/Web/HTML/Element/link for more information.
