Anchor

Constructor: Anchor(inner=None, role=None)

Name Type Description
inner obj the str or WebComponent object wrapped by the anchor;
role str the anchor role (this parameter is not used in a general scenario. It usually set by other WebComponent , which is using the anchor to introduce a specific action);

An Anchor is used for creating a hyperlink to pages, files, email addresses, locations on the same page, or other web-resources defined by a URL address. The Bootwrap also uses the Anchor in conjunction with other components, for example, in creating a navigation menu. 

Anchor('Google Search').link('https://www.google.com/')

link

The link(href) function connects the Anchor to another web-resource defined by the href parameter. 

Anchor('Google Search').link('https://www.google.com/')
Google Search

toggle

The toggle(wc) function makes a switch to a Panel defined the wc parameter. This function is typically called inside other WebComponent s such as Navigation for switching between tabs.

open

The open(wc) function opens a Dialog defined by the wc parameter. 

wc_quote = Dialog(
  "Steve Jobs Quote",
  "Sometimes life is going to hit you in " +
  "the head with a brick. Don’t lose faith."
)

Anchor("Steve Jobs Quote").toggle(wc_quote)
Steve Jobs Quote
Steve Jobs Quote
Sometimes life is going to hit you in the head with a brick. Don’t lose faith.

collapse

The collapse(wc) function expands or collapses a Panel defined by the wc parameter. 

wc_quote = Panel(
  "Sometimes life is going to hit you in " +
  "the head with a brick. Don’t lose faith."
).as_collapse()

Anchor("Steve Jobs Quote").collapse(wc_quote)
Steve Jobs Quote
Sometimes life is going to hit you in the head with a brick. Don’t lose faith.

dismiss

The dismiss() function is used in conjunction with a Dialog to initiate the close action. 

wc_quote = Dialog(
  "Steve Jobs Quote",
  "Sometimes life is going to hit you in " +
  "the head with a brick. Don’t lose faith.",
  Anchor("Close").dismiss()
)

Anchor("Steve Jobs Quote").open(wc_quote)
Steve Jobs Quote
Steve Jobs Quote
Sometimes life is going to hit you in the head with a brick. Don’t lose faith.
Close

submit

The submit() function is used in conjunction with a Form to initiate the submit-action.

Note: an Anchor associated with the submit-action will be displayed as a Button .

Styles

Use the following functions as_primary() , as_secondary() , as_success() , as_warning() , as_danger() , as_info() , as_light() , and as_dark() for changing the Anchor appearance.

Follow the Bootstrap guideline to applying these styles.

Button

Constructor: Button(name)

Name Type Description
name str the button name

A Button is used for performing interactions on a website.

Use the link(href) function to connect the Button to another web-resource defined by the href parameter. In this scenario, the Button behaves similar to the Anchor . 

# the button without style
Button('Google Search').link('https://www.google.com/')
Google Search

Button Styles

Use the following functions as_primary() , as_secondary() , as_success() , as_warning() , as_danger() , as_info() , as_light() , and as_dark() for changing the Button appearance.

Use the as_outline() function to make the Button without background. Note, the Button will have an outline look only if you apply one of the predefined styles. For the Button without style, this function will have no effect.

Follow the Bootstrap guideline to applying these styles. 

# to make a different appearance
Button('Primary').as_primary()
Button('Secondary').as_secondary()
Button('Success').as_success()
Button('Warning').as_warning()
Button('Danger').as_danger()
Button('Info').as_info()
Button('Light').as_light()
Button('Dark').as_dark()

# to make the outline look
Button('Primary').as_primary().as_outline()
Button('Secondary').as_secondary().as_outline()
Button('Success').as_success().as_outline()
Button('Warning').as_warning().as_outline()
Button('Danger').as_danger().as_outline()
Button('Info').as_info().as_outline()
Button('Light').as_light().as_outline()
Button('Dark').as_dark().as_outline()

Disable Button

But default the Button always enabled. Use the as_disabled() function to make the Button disabled. Disable status prevent user to initiate any action assigned to the Button . 

# the button without style
Button('Enabled').as_primary()
Button('Disabled').as_primary().as_disabled()

Dialog

Constructor: Dialog(title, content, *actions)

Name Type Description
title str the dialog title
content str|WC the content inside a dialog window, which can be either a str for a simple message or a WebComponent to provide more complex functionality
actions list the list of dialog actions, normally it should be Button s

A Dialog is a web-component to display a modal window. 

dialog = Dialog(
  'Greeting',
  'Hello World!',
  Button('Bye').dismiss()
)
button = Button('Say Hello').toggle(dialog)
Greeting
Hello World!

Question Dialog

An example of how Dialog can be used for confirming a specific action. Make sure that you provide a correct URL, which Confirm Button linked to. You can specify define this URL something like this local/file_system?file_id=1234567&action=delete 

dialog = Dialog(
  'Delete File',
  'Are you sure?',
  Button('Confirm').as_danger().link('url/to/act')
  Button('Cancel').dismiss()
).as_danger()

button = Button('Confirm').as_danger().toggle(dialog)
Delete File
Are you sure that you want to delete this file?

Complex Dialog

An example of how Dialog can be used for buying Google shares. It contains a web Form where a user can specify an amount (in Dollars) for buying shares. When the user presses the Buy Button , the form will be submitted to the server (at url/to/act ). If the user selects Cancel , the dialog will be closed, discarding the buying action. 

dialog = Dialog(
    'Google (NASDAQ: GOOGL)',
    Form(
        NumericInput(
            'Amount($)',
            'amount',
            placeholder=\
              'enter an amount for buying shares'
        ),
        Button('Cancel').\
          add_classes('float-right').\
          dismiss(),
        Button('Buy').\
          add_classes('float-right mr-2').\
          as_success().\
          submit()
    ).on_submit('url/to/act')
)

button = Button('Buy Shares').\
  as_primary().\
  toggle(dialog)
Google (NASDAQ: GOOGL)

Checkbox Input

Constructor: Checkbox(name, label)

Name Type Description
label str the input label showing for user
name str the input name, used for getting an input value from the form, during handling a submit action
checked bool True to make flag checked and False to make flag unchecked accordingly

A CheckboxInput is a component which can be in two states checked or unchecked .

The as_disabled() function prevents user to modify a status of this component. 

CheckboxInput('One', 'opt1')
CheckboxInput('Two', 'opt2', True)
CheckboxInput('Three', 'opt3').as_disabled()

Select Input

Constructor: SelectInput(name, label, *options)

Name Type Description
label str the input label showing for user
name str the input name, used for getting an input value from the form
options tuple the set of options

A SelectInput is a component for selecting a specific options from the dropdown list. Use SelectInput.Option(name, value, disabled=False) class to define selector options. The name parameter defines what user sees in the options list. The value parameters defines what is set if user selects a specific options. The disabled parameters makes an option unavailable for selection.

The function as_radio() changes the selector look from the dropdown list to a set of radio buttons.

The as_disabled() function prevents user to select any options. 

options = [
  SelectInput.Option('One', 1),
  SelectInput.Option('Two', 2),
  SelectInput.Option('Three', 3, disabled=True)
]

SelectInput('Selector1', 'choice', 2, options)
SelectInput('Selector2', 'choice', 2, options).as_disabled()
SelectInput('Selector3', 'choice', 2, options).as_radio()

Text Input

Constructor: TextInput(name, label, value=None, placeholder=None)

Name Type Description
label str the input label showing for user
name str the input name, used for getting an input value from the form
value str the value to set
placeholder str the text displayed when no value is entered

A TextInput is a component for entering a text information.

The with_multirows(n) function expands the text input to n rows. Use this option if the user needs to enter a long piece of text.

The for_email() and for_password() functions configure the input for a specific type of information ( email and password accordingly).

The as_disabled() function prevents user from entering any values. 

# the text input
TextInput('Text1', 'text')
TextInput('Text2', 'text').with_multirows(3)
TextInput('Text3', 'text', placeholder='type here')
TextInput('Text4', 'text', 'Hello World!')
TextInput('Text5', 'text').as_disabled()

# the email input
TextInput('Email', 'email', 'my@email.com').for_email()

# the password input
TextInput('Password', 'password', '********').for_password()

Constructor: List.Item(*items)

Navigation

Constructor: Navigation(*items)

Name Type Description
items tuple the navigation items.

A Navigation is a web-component for grouping other controls and providing an intuitive way to switch between these groups.

Use Navigation.Item(name, content, active=False) class to define navigation items. The name defines what user sees in the navigation list. The content defines what is displayed when a specific navigation item is selected. The active makes a navigation item content is visable for user (in other words it make this item selected).

By default Navigation is rendered in a form of anchors. To change the navigation style, apply the following functions as_tabs() , as_pills() and as_vertical() .

Use the as_tabs() function to make navigation looks like tabs.

Use the as_pills() function to make navigation looks like buttons.

Use the as_vertical() function to change navigation items orientation from horizontal to vertical. In the case of the vertical orientation, the navigation bar is displayed on the left and content on the right. 

Navigation(
  Navigation.Item('Chapter 1', 'Text 1', True),
  Navigation.Item('Chapter 2', 'Text 2'),
  Navigation.Item('Chapter 3', 'Text 3')
)

Navigation(
  Navigation.Item('Chapter 1', 'Text 1', True),
  Navigation.Item('Chapter 2', 'Text 2'),
  Navigation.Item('Chapter 3', 'Text 3')
).as_vertical()

Navigation(
  Navigation.Item('Chapter 1', 'Text 1', True),
  Navigation.Item('Chapter 2', 'Text 2'),
  Navigation.Item('Chapter 3', 'Text 3')
).as_tabs()

Navigation(
  Navigation.Item('Chapter 1', 'Text 1', True),
  Navigation.Item('Chapter 2', 'Text 2'),
  Navigation.Item('Chapter 3', 'Text 3')
).as_pills()

Navigation(
  Navigation.Item('Chapter 1', 'Text 1', True),
  Navigation.Item('Chapter 2', 'Text 2'),
  Navigation.Item('Chapter 3', 'Text 3')
).as_pills().as_vertical()

Javascript

Constructor: Javascript(src=None, script=None, submap=None)

Name Type Description
src str the URL to load Javascript code
script str the Javascript code
submap dict The map with substitutions, binding the Javascript with Python objects

A Javascript is a web-component for inserting Javascript code into the HTML page. This can be done in two ways. You can specify an external Javascript file via a URL using the src parameter. Or you can define Javascript code using the script parameter. To make a connection between Javascript and Python objects use the submap parameter. submap is a dictionary where key is a name of a variable within Javascript to replace with value - Python object. If Python object is an instance of WebComponent the substitution process with place it's identifier into the Javascript 

# Javascript URL
Javascript('https://ajax...0/jquery.min.js')

# Javascript code
label = Text('Answer:').as_strong()
answer = Text('unknown').add_classes('mr-2 ml-2')
btn_yes = Button('Yes').as_success()
btn_no = Button('No').as_danger()

action = Javascript(
  script='''
    $("#btn_yes").on("click",function(){
      $("#answer").text("Yes");
    });
    $("#btn_no").on("click",function(){
      $("#answer").text("No");
    });
  ''',
  submap={
    'answer': answer,
    'btn_yes': btn_yes,
    'btn_no': btn_no
  }
)

Panel(
  label, answer, btn_yes, btn_no, action
)
Answer: unknown
