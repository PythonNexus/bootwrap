Anchor

Constructor: Anchor(name)

Name Type Description
name str the anchor name

An Anchor is used for creating a hyperlink to pages, files, email addresses, locations on the same page, or other web-resources defined by a URL address. The Bootwrap also uses the Anchor in conjunction with other web-components, for example, in creating a navigation menu.

Use the link(href) function to connect the Anchor to another web-resource defined by the href parameter. 

Anchor('Search').link('https://www.google.com/')
Search

Anchor Styles

Use the following functions as_primary() , as_secondary() , as_success() , as_warning() , as_danger() , as_info() , as_light() , and as_dark() for changing the Anchor appearance.

Follow the Bootstrap guideline to applying these styles. 

Anchor('Primary').as_primary()
Anchor('Secondary').as_secondary()
Anchor('Success').as_success()
Anchor('Warning').as_warning()
Anchor('Danger').as_danger()
Anchor('Info').as_info()
Anchor('Light').as_light()
Anchor('Dark').as_dark()

Button

Constructor: Button(name)

Name Type Description
name str the button name

A Button is used for performing interactions on a website.

Use the link(href) function to connect the Button to another web-resource defined by the href parameter. In this scenario, the Button behaves similar to the Anchor . 

# the button without style
Button('Search').link('https://www.google.com/')
Search

Button Styles

Use the following functions as_primary() , as_secondary() , as_success() , as_warning() , as_danger() , as_info() , as_light() , and as_dark() for changing the Button appearance.

Use the as_outline() function to make the Button without background. Note, the Button will have an outline look only if you apply one of the predefined styles. For the Button without style, this function will have no effect.

Follow the Bootstrap guideline to applying these styles. 

# to make a different appearance
Button('Primary').as_primary()
Button('Secondary').as_secondary()
Button('Success').as_success()
Button('Warning').as_warning()
Button('Danger').as_danger()
Button('Info').as_info()
Button('Light').as_light()
Button('Dark').as_dark()

# to make the outline look
Button('Primary').as_primary().as_outline()
Button('Secondary').as_secondary().as_outline()
Button('Success').as_success().as_outline()
Button('Warning').as_warning().as_outline()
Button('Danger').as_danger().as_outline()
Button('Info').as_info().as_outline()
Button('Light').as_light().as_outline()
Button('Dark').as_dark().as_outline()
Primary Secondary Success Warning Danger Info Light Dark Primary Secondary Success Warning Danger Info Light Dark

Disable Button

But default the Button always enabled. Use the as_disabled() function to make the Button disabled. Disable status prevent user to initiate any action assigned to the Button . 

# the button without style
Button('Enabled').as_primary()
Button('Disabled').as_primary().as_disabled()
Enabled Disabled

Checkbox Input

Constructor: Checkbox(name, label)

Name Type Description
label str the input label showing for user
name str the input name, used for getting an input value from the form, during handling a submit action
checked bool True to make flag checked and False to make flag unchecked accordingly

A CheckboxInput is a component which can be in two states checked or unchecked .

The as_disabled() function prevents user to modify a status of this component. 

CheckboxInput('One', 'opt1')
CheckboxInput('Two', 'opt2', True)
CheckboxInput('Three', 'opt3').as_disabled()

Select Input

Constructor: SelectInput(name, label, *options)

Name Type Description
label str the input label showing for user
name str the input name, used for getting an input value from the form
options tuple the set of options

A SelectInput is a component for selecting a specific options from the dropdown list. Use SelectInput.Option(name, value, disabled=False) class to define selector options. The name parameter defines what user sees in the options list. The value parameters defines what is set if user selects a specific options. The disabled parameters makes an option unavailable for selection.

The function as_radio() changes the selector look from the dropdown list to a set of radio buttons.

The as_disabled() function prevents user to select any options. 

options = [
  SelectInput.Option('One', 1),
  SelectInput.Option('Two', 2),
  SelectInput.Option('Three', 3, disabled=True)
]

SelectInput('Selector1', 'choice', 2, options)
SelectInput('Selector2', 'choice', 2, options).as_disabled()
SelectInput('Selector3', 'choice', 2, options).as_radio()

Text Input

Constructor: TextInput(name, label, value=None, placeholder=None)

Name Type Description
label str the input label showing for user
name str the input name, used for getting an input value from the form
value str the value to set
placeholder str the text displayed when no value is entered

A TextInput is a component for entering a text information.

The with_multirows(n) function expands the text input to n rows. Use this option if the user needs to enter a long piece of text.

The for_email() and for_password() functions configure the input for a specific type of information ( email and password accordingly).

The as_disabled() function prevents user from entering any values. 

# the text input
TextInput('Text1', 'text')
TextInput('Text2', 'text').with_multirows(3)
TextInput('Text3', 'text', placeholder='type here')
TextInput('Text4', 'text', 'Hello World!')
TextInput('Text5', 'text').as_disabled()

# the email input
TextInput('Email', 'email', 'my@email.com').for_email()

# the password input
TextInput('Password', 'password', '********').for_password()

Numeric Input

Constructor: NumericInput(name, label, placeholder=None)

Name Type Description
label str the input label showing for user
name str the input name, used for getting an input value from the form
value int|float the value to set
placeholder str the text displayed when no number is entered.

A NumericInput is a component for entering a number. Its behavior is identical to the TextInput except additional validation of entering number. The NumericInput control in some web browsers may contain buttons helping to enter a number without a keyboard. 

NumericInput('Number', 'number')
NumericInput('Number', 'number', placeholder='your number')
NumericInput('Number', 'number', '123')
NumericInput('Number', 'number').as_disabled()

Hidden Input

Constructor: HiddenInput(name, value)

Name Type Description
name str the input name, used for getting an input value from the form
value obj the value to set

A HiddenInput is a component for hosting value within a form without showing it to a user. 

HiddenInput('token', '123')
